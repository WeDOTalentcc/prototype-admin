**Versão:** 1.17 — 2026-04-09

**Documentos relacionados**

- Fluxo de referência (stack Python / FastAPI): `docs/wsi/WSI_FLOW_PONTA_A_PONTA.md`

## Como manter este checklist

1. Ao implementar um item ❌ ou fechar um ⚠️, atualizar a linha e a data no topo.
2. Em dúvida sobre comportamento, preferir `WSI_METHODOLOGY_COMPLETE_v2.md` + código citado na coluna “Evidência” de `WSI_RAILS_COVERAGE_F1_F11.md`.
3. Mudanças na stack Python **não** entram aqui automaticamente — sincronizar manualmente quando o fluxo ponta a ponta de produto for único.
