## 💾 **COMO BAIXAR E USAR**

### **📥 DOWNLOAD DOS ARQUIVOS:**

#### **Opção 1 - Download Individual:**
1. Clique no arquivo desejado
2. Clique em "Raw" no GitHub
3. Ctrl+S para salvar localmente

#### **Opção 2 - Download da Pasta Completa:**
1. Clone o repositório: `git clone [repo-url]`
2. Navegue para: `plataforma-lia/docs/`
3. Todos os arquivos estarão disponíveis

#### **Opção 3 - ZIP Download:**
1. Click no botão "Code" no GitHub
2. Selecione "Download ZIP"
3. Extraia e navegue para `/docs/`

### **🔄 CONVERSÃO DE FORMATOS:**

#### **Markdown → PDF:**
```bash
# Usando pandoc
pandoc arquivo.md -o arquivo.pdf --pdf-engine=wkhtmltopdf

# Usando comando online
https://markdown-to-pdf.com/
```

#### **Markdown → PowerPoint:**
```bash
# Usando pandoc
pandoc arquivo.md -o apresentacao.pptx

# Ou use: https://md2pptx.com/
```

#### **Markdown → Word:**
```bash
pandoc arquivo.md -o documento.docx
```

#### **Markdown → HTML:**
```bash
pandoc arquivo.md -o pagina.html --standalone
```

#### **PowerPoint → Customização:**
```bash
# O arquivo .pptx já está pronto para uso
# Abra diretamente no PowerPoint, Google Slides ou Keynote
# Edite cores, logos e conteúdo conforme necessário
```
