#### C. Princípios de IA (Expandido)
```markdown
## IA Responsável - 5 Princípios

### 1. Diversidade
Nossa equipe de desenvolvimento é composta por pessoas de diferentes gêneros, raças, origens e experiências. Essa pluralidade ajuda a construir uma IA mais justa.

### 2. Anti-Discriminação
LIA é configurada para evitar discriminação, excluindo campos protegidos (gênero, idade, etnia, orientação sexual, religião, deficiência) de qualquer análise.

### 3. Transparência
Todas as decisões da IA são explicáveis. Candidatos podem solicitar explicação detalhada de sua avaliação a qualquer momento.

### 4. Privacidade & Segurança
Dados pessoais sensíveis nunca são usados para avaliação. Usamos apenas informações relevantes (experiência, habilidades, formação).

### 5. Responsabilidade
Temos um Comitê de Ética e Compliance que supervisiona o desenvolvimento e uso da IA, garantindo alinhamento com boas práticas.
```

#### D. FAQ (NOVA)
```markdown
## Perguntas Frequentes

### Para Candidatos

**Como a IA avalia meu currículo?**
A LIA analisa apenas informações objetivas como experiência profissional, formação acadêmica e habilidades técnicas. Dados pessoais como nome, idade, gênero não são considerados na avaliação.

**A IA decide se sou contratado?**
Não. A IA nunca toma decisões finais. Ela apenas organiza e pontua candidaturas para ajudar o recrutador humano, que sempre faz a decisão final.

**Posso pedir explicação da minha avaliação?**
Sim. Você tem direito à explicação de qualquer decisão automatizada. Solicite via Portal LGPD.

**Como exerço meus direitos sobre meus dados?**
Acesse nosso Portal LGPD em wedotalent.com.br/privacidade para solicitar acesso, correção, exclusão ou portabilidade dos seus dados.

### Para Empresas

**Quais certificações vocês possuem?**
Estamos em processo de certificação SOC 2 Type II (previsão: Junho 2026) e ISO 27001 (previsão: Dezembro 2026). Nossa conformidade LGPD está ativa.

**Como funciona a auditoria de viés da IA?**
Planejamos auditorias anuais com Warden AI, com primeira auditoria prevista para Q3 2026. Resultados serão publicados neste Trust Center.

**Vocês assinam DPA?**
Sim. Disponibilizamos Data Processing Agreement para clientes enterprise. Solicite via security@wedotalent.com.br.
```

## 6. Conclusão

### Principais Aprendizados

1. **Tezi AI** - Melhor estrutura técnica (via Vanta), controles detalhados, chat interativo
2. **Popp AI** - Melhor comunicação de "fairness", auditoria mensal de viés, links públicos
3. **Gupy** - Melhor linguagem didática, foco no candidato, DPO nominado, números de diversidade
4. **Findem** - Melhor cobertura multi-jurisdicional (GDPR, CCPA, CPRA, Colorado, OFCCP)
5. **hireEZ** - Melhor social proof com logos de clientes enterprise

### Recomendação Principal

O Trust Center WeDo Talent já possui **conteúdo sólido e completo**. As principais melhorias são:

1. **Forma sobre conteúdo** - Tornar mais visual (badges, ícones)
2. **Linguagem** - Simplificar para ser mais acessível
3. **Interatividade** - FAQ, Portal do Titular integrado
4. **Transparência** - DPO nominado, dashboard de AI Assurance
5. **Automação** - Migrar para Vanta/Drata quando SOC 2 estiver pronto
