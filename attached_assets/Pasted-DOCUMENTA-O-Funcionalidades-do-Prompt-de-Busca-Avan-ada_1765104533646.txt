DOCUMENTAÇÃO: Funcionalidades do Prompt de Busca Avançada
1. BOTÃO "SUGESTÕES" (Toggle)
Localização: Na barra de tags, abaixo do campo de busca

Funcionamento:

Estado: É um toggle on/off que controla as sugestões automáticas durante a digitação
Quando ativo: Mostra "Sugestões" com ícone Sparkles em cor #60BED1
Quando desativado: Mostra "Sugestões off" com ícone cinza
Backend: Chama a API /api/backend-proxy/search-assistant/autocomplete/?query={texto} enquanto o usuário digita (com debounce de 400ms)

Dados retornados:

{
  "items": [
    {
      "text": "texto sugerido",
      "category": "categoria",
      "icon": "ícone",
      "description": "descrição",
      "insert_text": "texto a inserir"
    }
  ]
}

Comportamento visual: Dropdown de autocomplete aparece abaixo do input com as sugestões

2. SUGESTÕES FIXAS (Cards de exemplo)
Localização: Abaixo da barra de tags, seção "Sugestões:"

Funcionamento:

São 4 cards fixos com exemplos de buscas completas
Ao clicar, preenche o campo de busca com o texto e dispara o parsing de entidades
Exemplos atuais:

"Backend Sênior em São Paulo, 5+ anos em fintechs, Node.js e Python"
"Product Manager Pleno remoto, experiência em B2B SaaS, metodologias ágeis"
"Data Scientist Sênior híbrido, 4+ anos em e-commerce, Python e ML"
"Tech Lead em Campinas, 7+ anos em startups, React e liderança de times"
3. BOTÃO "ASSISTENTE"
Localização: Na barra de tags, após o toggle de Sugestões

Visual: Botão preto com ícone Wand2 e texto "Assistente"

Funcionamento atual: Mostra tooltip explicativo sobre as funcionalidades:

Indica critérios faltantes
Sugere sinônimos e termos relacionados
Alerta sobre buscas muito amplas ou restritivas
4. ANÁLISE DE QUALIDADE DA BUSCA (Assistente em ação)
Localização: Aparece abaixo da barra de tags quando há texto no campo de busca

Backend: Duas chamadas paralelas:

/api/backend-proxy/search/parse-query/ - Extrai entidades (cargo, localização, experiência, etc.)
/api/backend-proxy/search-assistant/analyze/ - Analisa qualidade da busca
Elementos visuais:

4.1 Barra de Completude
Mostra: Porcentagem de qualidade (0-100%)
Cores: Verde (≥60%), Amarelo (40-59%), Vermelho (<40%)
Ação recomendada: Texto como "Adicione localização" ou "Especifique experiência"
4.2 Tags de Critérios (Localização, Cargo, Experiência, Setor, Habilidades)
Cinza sem preenchimento: Critério não detectado na busca
Colorido com valor: Critério detectado (ex: "Localização · São Paulo")
Ícone de loading "Analisando..." enquanto processa
4.3 Alertas Inteligentes
Warning (âmbar): Alertas sobre buscas problemáticas
Info (azul #60BED1): Sugestões de melhoria
Cada alerta pode ter um botão clicável que adiciona o valor sugerido à busca
5. PROMPT ENHANCEMENT (Melhoria de Prompt)
Localização: Aparece acima do campo de busca quando há texto (mínimo 10 caracteres)

Backend: /api/backend-proxy/enhance-prompt (debounce de 1500ms)

Dados retornados:

{
  "enhanced_query": "busca melhorada",
  "explanation": "explicação da melhoria",
  "confidence": 0.85,
  "suggestions": [...]
}

Ações: Aceitar melhoria (substitui o texto) ou Dispensar

6. PARSING DE ENTIDADES
Entidades extraídas automaticamente:

job_title (Cargo)
location (Localização)
years_experience (Experiência)
industry (Setor)
skills[] (Habilidades - array)
seniority (Senioridade)
company (Empresa)
