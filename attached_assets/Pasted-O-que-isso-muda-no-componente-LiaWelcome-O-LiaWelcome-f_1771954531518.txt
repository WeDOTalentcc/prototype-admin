O que isso muda no componente LiaWelcome
O LiaWelcome fica muito mais simples — é basicamente uma variante visual do LiaMessage, mas com layout centralizado para quando o chat está vazio:

LiaWelcome (simplificado):
┌──────────────────────────────────────────────┐
│                                               │
│              ┌────────┐                       │
│              │  🧠    │                       │  ← Brain icon (maior que o normal)
│              │ (cyan) │                       │     48px vs 32px do avatar normal
│              └────────┘                       │
│                                               │
│     Olá! Eu sou a LIA, sua assistente        │  ← Título (Open Sans, text-lg)
│        de recrutamento.                       │
│                                               │
│   Posso te ajudar a criar vagas, buscar      │  ← Subtítulo (text-sm, gray-500)
│   candidatos e gerenciar processos.          │
│                                               │
│   Me conta: no que está trabalhando hoje?    │  ← Prompt conversacional
│                                               │
└──────────────────────────────────────────────┘
Props (reduzidas):
- title: string          (default: "Olá! Eu sou a LIA...")
- subtitle: string       (default: "Posso te ajudar a...")  
- prompt: string         (default: "Me conta: no que está trabalhando hoje?")
- icon: ReactNode        (default: Brain)
- variant: 'first-time' | 'new-conversation'
O variant: 'new-conversation' seria para quando não é a primeira vez — a LIA já conhece o recrutador:

Pelo que você descreveu, vocês já têm:

Suggestion chips abaixo ou acima do prompt (frases curtas clicáveis que preenchem o input)
Botão de ideias que abre uma lista de sugestões
