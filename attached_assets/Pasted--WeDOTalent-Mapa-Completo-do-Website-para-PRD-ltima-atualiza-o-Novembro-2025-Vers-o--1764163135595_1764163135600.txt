## 1. Visão Geral do Produto

**WeDOTalent** é uma plataforma de recrutamento AI-first brasileira que combina:
- **LIA** (Agente de IA Conversacional) - Assistente inteligente de recrutamento via WhatsApp/Teams
- **ATS Completo** - Sistema de Rastreamento de Candidatos como standalone ou plugin para ATSs existentes

**Proposta de Valor Principal:**
- Redução de 70% no trabalho manual
- Diminuição significativa do time-to-hire
- ROI mensurável
- WhatsApp-first para o mercado brasileiro
- "Service as Software" - tecnologia + suporte dedicado

## 6. Design System & Brandbook V2

### Identidade Visual Dual:

| Contexto | Aplicação | Estilo |
|----------|-----------|--------|
| **WeDO Institucional** | Páginas corporativas | 85% preto/branco/cinza, minimalista, Apple-inspired |
| **LIA (Produto IA)** | Páginas de produto | Azul claro (#74A8D2), formas orgânicas, gradientes fluidos |

### Cores Principais:
- **Preto:** Principal
- **Branco:** Contraste
- **Azul LIA:** #74A8D2 (elementos de IA)
- **Verde Suave:** #B4D3C1 (usar com moderação)
- **Ciano:** Cor de interação (hover/active)

### Tipografia:
- Sentence case em todo o site (exceto nomes próprios e siglas)
- Font Display para títulos
- Font Body para texto

### Animações:
- Fade in/slide up para entrada de elementos
- Efeito typewriter para manifestos
- Pulso suave em CTAs (scale 1→1.05→1, 2s)
- Gradientes animados em backgrounds (20s ciclo)

### Acessibilidade:
- WCAG 2.1 Level AA
- `prefers-reduced-motion` suportado
- Navegação por teclado
- Suporte a screen readers
